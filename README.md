# Automated Loan Processing & Risk Assessment System

AI-powered loan eligibility and risk assessment system built using FastAPI, React, and Machine Learning.
The system evaluates loan applications without relying on third-party credit bureau APIs.

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React + Tailwind + Vite
- ML: Scikit-learn, XGBoost
- Database: PostgreSQL
- Explainability: SHAP
